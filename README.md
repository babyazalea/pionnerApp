# PIONNER APP

~https://babyazalea.github.io/pionnerApp/~

* React JS 기반의 웹 어플리케이션.

* News API를 통해 TOP 헤드라인을 15개 불러오고 그것을 자동으로 보여준다.

* 오른쪽 상단의 토글을 움직여서 특정 언론사의 기사를 가릴 수 있다.(투명도로 제어)

* 필터링 부분이 완벽하지 않아서 아쉽다.
